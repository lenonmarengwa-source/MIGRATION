# Zimbabwe Bayesian Migration Risk Model

**Advanced Hierarchical Spatio-Temporal CAR Model**

![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?style=for-the-badge&logo=streamlit&logoColor=white)
![PyMC](https://img.shields.io/badge/PyMC-3776AB?style=for-the-badge&logo=python&logoColor=white)

## Features
- Hierarchical Bayesian model with CAR spatial effects
- Interactive Kepler.gl maps
- User authentication
- Fast inference with JAX/NumPyro
- Synthetic data for out-of-the-box demo
- Docker + CI/CD ready

## Quick Start
```bash
git clone https://github.com/lenonmarengwa-source/MIGRATION.git
cd MIGRATION
pip install -r requirements.txt
streamlit run app.py
```
